# Employee Attrition Prediction using Logistic Regression

## Project Overview
This project predicts whether an employee is likely to leave the company using Logistic Regression. The IBM HR Analytics Employee Attrition dataset is used for training and testing the model.

## Dataset
IBM HR Analytics Employee Attrition & Performance Dataset

## Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Project Workflow
1. Import Libraries
2. Load Dataset
3. Exploratory Data Analysis (EDA)
4. Data Cleaning
5. Encode Categorical Variables
6. Train-Test Split
7. Feature Scaling
8. Logistic Regression Model
9. Model Evaluation
10. Confusion Matrix
11. ROC-AUC Curve
12. Conclusion

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-Score
- Confusion Matrix
- ROC-AUC Score

## Conclusion
The Logistic Regression model successfully predicts employee attrition and provides insights into factors affecting employee turnover.
